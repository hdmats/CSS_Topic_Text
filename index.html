CSS TOPIC: TEXT

text-align

Align to the center, right, left (just like in Word or Google Doc), or justify (alters spacing so every line ends at the same point).

See the Pen text-align CSS by Henry Matsen (@hdmats) on CodePen.

text-decoration

text-decoration is about adding lines. The options are overline, through-line, or underline. Their are also wavy, dotted and double-line styles.

See the Pen text-decoration CSS by Henry Matsen (@hdmats) on CodePen.

text-indent

text-indent length can be set either by percentage(%), pixels(px), or by any other standard measurement (em). The indent can be both positive and negative.

See the Pen text-indent CSS by Henry Matsen (@hdmats) on CodePen.

text-transform

text-transform can either convert all letters to UPPERCASE, lowercase, and capitalize every word (captialize).

See the Pen text-transform CSS by Henry Matsen (@hdmats) on CodePen.

line-height

line-height can change the height of the line either by a number, a percentage(%), or by pixels(px). If using a percentage or number be aware that it multiplies it by the font size so a larger font would increase the line height.

See the Pen line-height CSS by Henry Matsen (@hdmats) on CodePen.
